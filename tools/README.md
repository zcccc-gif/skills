# 工具类

扩展 AI 能力的工具技能。

## 技能

| 技能 | 说明 | 触发关键词 |
|------|------|-----------|
| `skill-creator-zh` | 创建和测试新技能 | "创建技能"、"新建技能" |
| `find-skills-zh` | 搜索和安装现有技能 | "找技能"、"搜索skill" |
| `theme-factory-zh` | 应用专业主题样式 | "主题"、"配色"、"样式" |

## 使用示例

### skill-creator-zh
```
你：我想创建一个新的技能
AI：[引导你完成技能创建流程]

### find-skills-zh

AI：搜索中...
     找到了！image-processing-skill
     安装命令：npx skills add xxx
```

