# 沟通效率类

提升沟通效率，减少 token 消耗的技能。

## 技能

| 技能 | 说明 | 触发关键词 |
|------|------|-----------|
| `caveman-zh` | 极简沟通，减少 75% token | "原始人模式"、"简洁点" |
| `grill-me-zh` | 深入追问直到所有决策明确 | "追问"、"压力测试" |

## 使用示例

### caveman-zh
```
你：原始人模式
AI：Bug in auth. Token check use < not <=. Fix:
```

### grill-me-zh
```
你：追问一下我的登录方案
AI：## 问题 1
     你用什么认证方式？OAuth/邮箱密码/手机号？
你：邮箱密码
AI：## 问题 2
     密码存储用什么？bcrypt/argon2？
...
```
