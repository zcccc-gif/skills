# 画布设计技能

这些是创建设计理念的指导 - 美学运动随后通过视觉表达。仅输出 .md 文件、.pdf 文件和 .png 文件。

分两步完成：
1. 设计理念创建（.md 文件）
2. 通过在画布上创建来表达（.pdf 或 .png 文件）

## 设计理念创建

首先创建一个**视觉哲学**（非布局或模板），通过以下方式诠释：
- 形式、空间、颜色、构图
- 图像、图形、形状、图案
- 极少文字作为视觉点缀

### 关键理解
- **收到什么**：用户的微妙输入或指示，作为基础而非约束
- **创建什么**：设计理念/美学运动
- **接下来**：同一版本接收理念并**视觉表达** - 创建 90% 视觉设计、10% 必要文字的作品

### 如何生成视觉理念

**命名运动**（1-2 个词）：
- "粗野诗意" / "色彩语言" / "冥想空间" / "有机系统" / "几何寂静"

**阐述理念**（4-6 段 - 简洁但完整）：

表达理念如何通过以下方式体现：
- 空间与形式
- 颜色与材料
- 尺度与节奏
- 构图与平衡
- 视觉层次

**关键指南：**
- **避免冗余**：每个设计方面只提及一次
- **反复强调工艺**：最终作品应看起来花费了无数小时精心打造
- **留出创意空间**：具体说明美学方向，但保持简洁

### 理念示例

**"混凝土诗学"**
理念：通过纪念性形式和大胆几何进行视觉交流。
视觉表达：巨大的色块、雕塑般的字体（巨大的单词、微小的标签）、粗野主义的空间划分。

**"色彩语言"**
理念：颜色作为主要信息系统。
视觉表达：几何精确，色彩区域创造意义。字体极简 - 小的无衬线标签让色彩场来交流。

**"冥想空间"**
理念：通过纹理和呼吸空间进行安静的视觉沉思。
视觉表达：纸张纹理、墨水渗透、大量负空间。摄影和插图占主导。字体低语般（小型、克制、服务视觉）。

### 基本原则
- **视觉理念**：创建通过设计表达的美学世界观
- **极少文字**：文字稀疏、必要、作为视觉元素整合
- **空间表达**：思想通过空间、形式、颜色、构图交流
- **艺术自由**：下一个 Claude 视觉诠释理念 - 提供创意空间
- **纯粹设计**：这是制作艺术对象，不是装饰的文档
- **精湛工艺**：反复强调最终作品必须看起来精心打造

**设计理念应为 4-6 段长。** 输出为 .md 文件。

## 推断微妙参考

**关键步骤**：在创建画布之前，从原始请求中识别微妙的概念线索。

**基本原则**：
主题是**嵌入艺术本身中的微妙、小众参考** - 并非总是字面的，总是精致的。熟悉主题的人应该直觉感受到，而其他人只是体验一个大师级的抽象构图。

## 画布创建

理念和概念框架确立后，在画布上表达。使用设计理念和以下指导创作一件精品。

**重要**：对于任何类型的内容，方法都应该精致。永远不要失去这是艺术而非卡通或业余作品的意识。

### 创作要求

- 使用重复图案和完美形状
- 将抽象哲学设计视为科学圣经，借用系统观察的视觉语言
- 稀疏的排版和系统性的参考标记
- 用简单的短语或细节微妙地定位作品
- 使用有限但有意图的配色方案

### 文字作为上下文元素

文字总是极少且视觉优先，但让上下文引导：
- 朋克海报可能有更大、更激进的字体
- 极简陶艺工作室标识可能更安静

**重要：使用不同字体。搜索 `./canvas-fonts` 目录。精致是不可妥协的。**

下载并使用所需的任何字体。让字体成为艺术本身的一部分。

### 最终步骤

**重要**：用户已经说过"它不够完美。它必须是原始的、工艺的杰作，仿佛即将在博物馆展出。"

**关键**：精炼作品时，避免添加更多图形；而是精炼已创建的内容，使其极其精致。

再次审视代码，进一步精炼/打磨，使其成为哲学设计的杰作。

输出最终结果为单个可下载的 .pdf 或 .png 文件，以及使用的理念作为 .md 文件。
