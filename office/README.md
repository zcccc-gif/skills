# 办公文档类

处理 Office 文档的技能。

## 技能

| 技能 | 说明 | 触发关键词 |
|------|------|-----------|
| `docx-zh` | Word 文档创建、编辑 | "word"、"docx"、"文档" |
| `pptx-zh` | PPT 演示文稿制作 | "ppt"、"演示"、"幻灯片" |
| `xlsx-zh` | Excel 电子表格处理 | "excel"、"表格"、"xlsx" |
| `pdf-zh` | PDF 文件处理 | "pdf"、"读取pdf" |

## 使用示例

```
你：帮我创建一个 Word 文档包含目录和标题
AI：[使用 docx-zh 技能创建专业的 Word 文档]

你：制作一个 10 页的 PPT 演示
AI：[使用 pptx-zh 技能创建专业的演示文稿]

你：处理这个 Excel 文件，添加公式计算总和
AI：[使用 xlsx-zh 技能处理电子表格]
```
