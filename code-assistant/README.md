# 代码助手类

智能代码修改、测试驱动开发、调试诊断技能。

## 技能

| 技能 | 说明 | 触发关键词 |
|------|------|-----------|
| `code-assistant` | 修改代码前先给方案，用户确认后再执行 | "修改代码"、"重构"、"优化代码" |
| `tdd-zh` | 测试驱动开发，红-绿-重构循环 | "tdd"、"红绿重构"、"测试优先" |
| `diagnose-zh` | 严格的调试流程 | "诊断"、"调试"、"debug" |

## 使用示例

### code-assistant
```
你：帮我把 calc() 函数改名叫 calculate_total_price()
AI：## 修改方案
     **目标：** 重命名函数使其更清晰
     **步骤：** 1. 重命名函数定义 2. 更新所有调用处
     **请选择：** 1. 执行 2. 跳过 3. 补充信息
你：1
AI：## 修改完成
     ✅ calc → calculate_total_price
```

### tdd-zh
```
你：用 tdd 写一个用户登录功能
AI：## RED 阶段
     写测试：用户可以用正确密码登录
     ...
你：完成
AI：## GREEN 阶段
     实现登录逻辑...
```

### diagnose-zh
```
你：程序报错了
AI：## 阶段 1：构建反馈循环
     请提供复现步骤...
```
